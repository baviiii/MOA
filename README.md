

## Features

- Driver profile management
- Route tracking and optimization
- Admin dashboard
- Real-time updates
- Secure authentication

## Development

To run the project locally:

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
